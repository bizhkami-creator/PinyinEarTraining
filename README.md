# ピンイン耳トレ - HSK1級対応

- HSK1級単語 約150語に対応
- 苦手単語のみを抽出して練習できる復習モード搭載
- 苦手単語の一覧表示機能（弱点の可視化）
- 中国語音声の自動再生 (TTS)
- ピンイン＋声調のクイズ (3択/5択)
- スコアランキング表示
- 効果音（正解・不正解）設定可能
- アプリアイコン（Adaptive Icon）対応
