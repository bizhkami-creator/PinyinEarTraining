# ピンイン耳トレ - HSK1級対応

## 概要

HSK1級レベルの中国語音声を聞き、正しいピンインと声調を選ぶクイズアプリです。

## 主な機能

- HSK1級単語 約150語に対応
- 中国語音声の自動再生（TTS）
- ピンイン＋声調の3択 / 5択クイズ
- 正解・不正解表示および「あなたの回答」の表示
- 正解・不正解時の効果音（ON/OFF設定可能）
- 苦手単語の自動記録（間違えた回数をカウント）
- 苦手単語復習モード（ランキング保存対象外）
- 苦手克服機能（復習モードで正解を重ねるとリストから自動削除）
- 進捗状況の視覚的表示（進捗バー）
- 漢字・ピンイン・日本語意味の表示
- スコア・ランキング表示（TOP 10）

## 対象ユーザー

- 中国語初心者
- HSK1級学習者
- ピンインと声調を聞き分けたい学習者

## 開発環境

- Android Studio
- Kotlin
- XML Layout
- TextToSpeech
- SharedPreferences
