# 要件定義

## アプリ名

ピンイン耳トレ - HSK1級対応

## 目的

HSK1級レベルの中国語単語を対象に、音声を聞いて正しいピンインと声調を選ぶ練習ができるアプリを作る。

## 対象範囲

### 対象

- HSK1級必須単語 約150語
- ピンイン
- 声調
- 日本語意味
- 中国語音声読み上げ

### 対象外

- HSK2級以上
- 発音採点
- ユーザーアカウント
- オンライン同期
- 広告
- 課金

## 画面

### 1. ホーム画面

- 過去ランキングを降順で表示
- 最高スコアを表示
- 苦手単語数を表示
- 問題数を選択
- 3択 / 5択を選択
- 効果音のON/OFFを選択
- スタートボタンを表示
- 苦手単語復習ボタンを表示（苦手単語がある場合のみ動作）

### 2. クイズ画面

- 問題番号を表示
- クイズの進捗バーを表示
- 中国語音声を自動再生
- 再生ボタンを表示
- ピンイン選択肢を表示
- 回答後、結果画面へ進む

### 3. 回答結果画面

正解時：

- 正解メッセージ
- 正解音を再生
- 復習モード時は mistakeCount を減らし、0になったら苦手リストから削除
- 削除時は「苦手克服！」等のフィードバックを表示
- 漢字
- ピンイン
- 日本語意味

不正解時：

- 不正解メッセージ
- 不正解音を再生
- 苦手単語として保存（mistakeCountを更新）
- 正しいピンイン
- 漢字
- 日本語意味

### 4. スコア画面

- 正解数
- 正答率
- スコア
- ランキング順位（通常モードのみ）
- 達成日時（通常モードのみ）
- 復習モード時は、ランキング非保存の注釈を表示


### 5. ランキング画面

- 過去スコアTOP10
- もう一度遊ぶ
- ホームへ戻る

## データ項目

```text
level
hanzi
pinyin
meaning
mistakeCount (苦手単語管理用)
